# Net Square's Bambdas Script

A curated collection of powerful Bambda scripts for Burp Suite Repeater custom actions to enhance penetration testing workflows and automate repetitive tasks during manual security assessments.

## What are Bambdas?

Bambdas are lightweight, reusable Java-based scripts that run directly from Burp Suite's interface. They enable security testers to quickly personalize various tasks without building full extensions. Custom actions specifically are scripts that run in Burp Repeater to extract, transform, and analyze data during manual testing.

### Key Features
- Execute directly from Repeater interface without external tools
- Access to Burp's Montoya API for powerful request/response manipulation
- Can fetch data, modify requests before sending, and analyze responses
- Support for AI-powered features for smarter workflows 
- Reusable across projects through Bambda library

## Scripts 
- 403-bypass
- Burp Quick Fuzzer
- Request Attacker
- HTTP Method Fuzzer
- HTTPS Downgrade Attack Tester
- Timestamp Generate Replace
