# Http Method Fuzzer - Custom Repeater Action (Bambda Script)

**HTTP Method Fuzzer** is a Burp Suite Bambda script for Custom Repeater Action that lets you instantly fuzz for supported HTTP methods for particular request.

1. Clone this repo.
2. Open Burp Suite.
3. Navigate to **Extender → Bambda Scripts**.
4. Click **"Load"**, select `http-method-fuzzer.bambda`.
5. Set Location as `Custom Action` and `Repeater`, see the video
6. Select a request in **Repeater** context.
7. Run the script to send all payloads and view responses.
